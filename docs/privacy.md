# 隐私说明

## 本地优先

本项目 v0 默认本地运行。Python 脚本读取本地文件并写入本地 `outputs/` 目录；网页 demo 在浏览器本地生成 Markdown，不接后端，不接 API key。

## 不默认上传数据

项目不会默认上传活动信息、转写文本、人物记录或跟进任务。用户如果自行部署、分享或提交文件，需要自行检查是否包含私人信息。

## 不处理未经授权录音

请不要输入、保存或分析未经授权的录音、转写或聊天记录。录音/转写功能默认关闭，只处理用户主动提供且有权处理的文本。

## 不做人脸识别

本项目不做人脸识别，不根据照片、视频或外貌识别参会者。

## 不做声纹识别

本项目不做声纹识别，不根据声音判断说话人身份。

## 不推断私人敏感信息

不要从活动材料、转写文本或普通寒暄中推断私人敏感信息，包括但不限于健康、财务、家庭、政治、宗教、性取向等信息。

## 不自动发送跟进消息

项目只生成跟进话术草稿和任务清单，不自动发送邮件、微信或任何消息。

## 用户需要对输入内容负责

用户需要确认输入内容的来源、授权和必要性。涉及多人对话时，只应提取与活动目标相关的信息，并避免暴露无关私人内容。

## 公开发布 demo 的注意事项

公开发布 GitHub Pages demo、README、示例文件或截图时，不要放真实活动复盘、真实人物记录、联系方式、聊天记录或未授权转写。
