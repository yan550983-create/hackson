# 网页 Demo 使用说明

## 网页 demo 用法

进入 `web/index.html` 后，可以在浏览器本地填写活动信息，点击“生成会前简报”，右侧会显示 Markdown 风格的会前简报草稿。

## 示例活动说明

点击“加载示例活动”会填入虚构活动“AI 出海青年科创分享会”。示例中的嘉宾、机构和转写片段均为虚构数据，不包含真实个人隐私。

## 如何生成会前简报

1. 填写活动名称、链接、时间、城市、场地、主办方、主题、嘉宾、简介和用户目标。
2. 选择精力状态：低、中或高。
3. 点击“生成会前简报”。
4. 查看右侧 Markdown 预览。

## 如何生成会后复盘

1. 在“活动后转写复盘”区域填写活动名称。
2. 粘贴用户主动提供的转写文本或会后笔记。
3. 点击“生成复盘”。
4. 查看结构化复盘草稿。

## 如何复制 Markdown

点击输出区域右上角的“复制 Markdown”。浏览器会把当前会前简报或会后复盘复制到剪贴板。

## 如何下载 Markdown

点击输出区域右上角的“下载 Markdown”。文件名会包含日期、活动名和类型，例如 `2026-06-01-ai-event-pre-event-brief.md`。

## 当前 demo 的限制

- 不联网搜索。
- 不做真实音频上传或语音识别。
- 不做自动嘉宾资料识别。
- 不做登录、数据库或云端保存。
- 不自动发送任何跟进消息。
- 只做规则化草稿生成，复杂判断需要用户人工复核。
